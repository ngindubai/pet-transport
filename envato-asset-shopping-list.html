Image Optimisation Notes

Before Adding to Repo

  1. Resize: No image needs to be larger than 1920px wide. Hero images: 1920x900. Card thumbnails: 800x600. Logos: vector SVG preferred.
  2. Compress: Run through TinyPNG, Squoosh, or ImageOptim. Target: hero images under 200KB, card images under 80KB.
  3. Format: JPG for photographs. PNG for logos and illustrations with transparency. WebP optional (with JPG fallback) for modern browsers.
  4. Alt text: Every <img> in the Hugo templates has an alt attribute. Update these when replacing images to describe the actual content for accessibility and SEO.
  5. Lazy loading: All images below the fold should use loading="lazy". Hero images use loading="eager". This is already set in the Hugo templates.

Envato Elements Download Tips

  • Download the largest resolution available, then resize locally. Envato images often come at 5000px+.
  • Check the licence: Envato Elements licence covers web use on one project. All downloads under one subscription are covered.
  • Use the "Similar" feature after finding one good image to find matching style/photographer for visual consistency.
  • For city skylines (P4/P5), search by city name + "aerial" or "skyline" for the cleanest jumbotron-style images.
  • For pet transport specifics, try: "pet travel", "dog carrier", "cat crate", "animal transport", "pet relocation".
