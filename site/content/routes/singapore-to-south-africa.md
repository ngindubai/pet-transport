    - heading: "Starting both permit processes in parallel"
      body: |
        The DALRRD import permit and the AVS export licence have different processing timescales. DALRRD takes 4-8 weeks; AVS takes 2+ weeks. Start both at the same time rather than sequentially to avoid adding weeks to your timeline.

        If a titre test is required, that process (vaccination, blood draw 30+ days post-vax, possible waiting period) must be completed before the health certificate can be finalised. In that case the titre test work starts first, and the permit applications run in parallel during the titre test sequence.
