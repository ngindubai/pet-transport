Build and Deploy Reference

Standard Build (run from repo root)

# Full build + sitemap split:
build.bat

cd site
hugo --gc --minify
cd ..
python split_sitemap.py

# Then upload site\public\ to Hostinger

Post-Upload Checklist

  • Upload site/public/ contents to Hostinger root (not the folder itself)
  • Verify https://pettransportglobal.com/ loads correctly
  • Verify https://pettransportglobal.com/sitemap.xml returns sitemap index XML
  • Verify https://pettransportglobal.com/sitemap-routes.xml returns route URLs
  • Verify https://pettransportglobal.com/robots.txt contains Sitemap directive
  • Check one P1 route page renders: /pet-transport/united-kingdom-to-australia/
  • Check one blog article renders: /blog/uk-to-australia-pet-transport-complete-guide/
  • Submit all sitemaps in Google Search Console

Next Steps After Deployment

  1. Upload to Hostinger — upload site/public/ via file manager
  2. Submit sitemaps to GSC — all 4 section sitemaps + index
  3. Request indexing — URL Inspection for top 15 P1 routes + home page
  4. Task 3.6 (Content upgrades) — review P1 routes for content depth opportunities once ranking data arrives
  5. Task 3.8 (Gate Review) — run the 20-page content sample check (table above), complete performance review at 30 days
